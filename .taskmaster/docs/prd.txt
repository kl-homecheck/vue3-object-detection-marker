# Vue3 Object Detection Marker - 프로젝트 요구사항 문서

## 프로젝트 개요
Vue3를 사용한 사진별 object detection을 위한 npm 패키지 개발. 사용자가 이미지 위에 격자 형태로 영역을 선택하고 마킹할 수 있는 도구를 제공합니다.

## 핵심 기능 요구사항

### 1. 이미지 입력 지원
- URL 형태의 이미지
- Base64 인코딩된 이미지
- Blob 객체 형태의 이미지
- File 객체 형태의 이미지

### 2. 격자 시스템
- 244x244 픽셀 크기의 격자 그리드
- 이미지 크기에 따른 적응형 격자 배치
- 격자 경계선 시각적 표시

### 3. 선택 모드
#### 모드 1: 포인트 기반 선택
- 마우스 드래그 또는 터치 드래그
- 드래그 포인트 주변 픽셀들 자동 선택
- 연속적인 영역 선택 가능

#### 모드 2: 사각형 영역 선택
- 모드 전환 기능
- 드래그 시작점부터 끝점까지 사각형 영역 생성
- 사각형 내부 모든 격자 선택

### 4. 지우개 기능
- 드래그한 범위 주변 항목들 선택 해제
- 선택된 영역에서 특정 부분 제거
- 정밀한 영역 조정 가능

### 5. 시각적 피드백
- 선택된 격자 영역 하이라이트
- 격자 경계선 표시
- 실시간 선택 상태 반영
- 선택/해제 상태 구분 가능한 색상 시스템

## 기술적 요구사항

### 프론트엔드
- Vue 3 Composition API
- TypeScript 지원
- 반응형 디자인 (데스크톱/모바일)
- Canvas 또는 SVG 기반 렌더링

### 패키징
- npm 패키지로 배포
- Vue 3 컴포넌트 형태 제공
- 독립적인 라이브러리로 사용 가능

### 성능 요구사항
- 대용량 이미지 처리 최적화
- 실시간 인터랙션 지원
- 메모리 효율적인 격자 관리

### 기본 기능 시연
- 이미지 업로드/URL 입력
- 격자 표시 및 선택 기능

- 직관적인 도구 모음
